83  * 1. Set !OE low to enable output of encoder
84  * 2. Set SEL low to get high byte
85  * 3. Wait about 20 microseconds
86  * 4. Read MSB
87  * 5. Set SEL high to get low byte
88  * 6. wait about 20 microseconds

98  REG_PIOD_CODR |= (1<<SEL);
